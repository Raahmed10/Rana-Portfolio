# Rana Portfolio

Personal portfolio website built with **Laravel + Vue 3 + TailwindCSS**.

A modern, responsive portfolio showcasing my skills, projects, certificates, and contact information with clean UI and smooth interactions.

---

## Features
- Fully responsive design  
- Arabic / English language toggle  
- Certificates modal popup  
- Projects showcase section  
- Contact section  
- Smooth scroll & back-to-top button  
- Clean UI built with TailwindCSS  

---

## Tech Stack
- **Backend:** Laravel  
- **Frontend:** Vue 3 (Composition API)  
- **Styling:** TailwindCSS  
- **Build Tool:** Vite
